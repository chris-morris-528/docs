First, make sure you've completed the previous steps to setup a new OAuth application and that you have generated a client secret. If you haven't, follow the guide here first: [Authentication](https://developer.vanta.com/docs/api-access-setup)

For this example, we are going to retrieve a list of our Integrations that we have connected to our Vanta instance. To perform this request, you will need to have already created an application using the "Manage Vanta" type. (The "Build Integrations" application type will not work for this example)

The integrations endpoint is: [https://api.vanta.com/v1/integrations](https://api.vanta.com/v1/integrations). We will pass in our `Authorization` header and set the value to `Bearer` followed by your API token. Here is an example cURL to get you started (make sure to replace *your\_token\_here* with your actual API token):

<CodeGroup>
  ```bash cURL
  curl --location 'https://api.vanta.com/v1/integrations?pageSize=10' \
  --header 'Accept: application/json' \
  --header 'Authorization: Bearer _your_token_here_'
  ```

  ```JavaScript JavaScript - Fetch
  const myHeaders = new Headers();
  myHeaders.append("Accept", "application/json");
  myHeaders.append("Authorization", "Bearer _your_token_here_");

  const requestOptions = {
    method: "GET",
    headers: myHeaders,
    redirect: "follow"
  };

  fetch("https://api.vanta.com/v1/integrations?pageSize=100&pageCursor=<string>", requestOptions)
    .then((response) => response.text())
    .then((result) => console.log(result))
    .catch((error) => console.error(error));
  ```

  ```JavaScript NodeJS - Axios
  const axios = require('axios');

  let config = {
    method: 'get',
    maxBodyLength: Infinity,
    url: 'https://api.vanta.com/v1/integrations?pageSize=100&pageCursor=<string>',
    headers: {
      'Accept': 'application/json',
      'Authorization': 'Bearer _your_token_here_'
    }
  };

  axios.request(config)
  .then((response) => {
    console.log(JSON.stringify(response.data));
  })
  .catch((error) => {
    console.log(error);
  });
  ```

  ```Python Python - Requests
  import requests

  url = "https://api.vanta.com/v1/integrations?pageSize=100&pageCursor=<string>"

  payload = {}
  headers = {
    'Accept': 'application/json',
    'Authorization': 'Bearer _your_token_here_'
  }

  response = requests.request("GET", url, headers=headers, data=payload)

  print(response.text)
  ```
</CodeGroup>
